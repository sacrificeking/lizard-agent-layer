# Memory Policy

Default mode: curated memory.

## Commit-safe memory

- Stable user preferences.
- Accepted engineering lessons.
- Architectural decisions.
- Project workflow rules.

## Private memory

- Raw episodic logs.
- Tool traces.
- Local dashboards.
- Provider responses.
- Research drafts with sensitive context.

## Forbidden memory

- Secrets and API keys.
- Private keys.
- Raw customer data.
- Authentication tokens.
- Unredacted production data.
