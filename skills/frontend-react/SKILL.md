# Frontend React

## Rules

- Follow existing component patterns.
- Keep state ownership clear.
- Avoid broad refactors while fixing narrow behavior.
- Add focused tests for shared logic or risky UI behavior.
- Run typecheck and relevant tests before finalizing.
