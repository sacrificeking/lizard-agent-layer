# Roadmap

## Phase 1 - MVP

- Repo structure
- Profile schema
- Minimal, standard, and Supabase React finance profiles
- Codex adapter
- Initial reusable skills
- PowerShell installer with preview/apply modes

## Phase 2 - Safety and validation

- Installer idempotency checks
- Config validator
- Scratch-target smoke test
- Manifest sync command
- Upgrade preview command

## Phase 3 - Project import

- Analyze a target project and recommend a profile
- Generate project-specific skill selection
- Create merge suggestions for existing AGENTS.md

## Phase 4 - Optional data layer

- Local agent activity summaries
- Release-readiness dashboard
- Skill usage reports
- Private, gitignored exports

## Phase 5 - Multi-harness adapters

- Claude Code
- Cursor
- Gemini
- Generic AGENTS.md fallback
